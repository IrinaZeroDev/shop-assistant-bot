Разработка многофункционального чат-бота с интеграцией GigaChat и ProxyAPI
Активный

Ваша задача — создать чат-бота, который будет эффективно взаимодействовать с клиентами, 
используя современные API и обеспечивать стабильную работу через прокси-серверы.

Задачи:

    Разработать чат-бота на Python и Node.js с использованием GigaChat API и ProxyAPI
    Реализовать REST API для взаимодействия с внешними сервисами
    Обеспечить корректную обработку типовых вопросов клиентов
    Настроить сценарии диалогов согласно требованиям
    Собрать и структурировать данные для анализа взаимодействий

Требования:

    Опыт работы с Python и Node.js
    Знание GigaChat API и ProxyAPI
    Умение создавать и использовать REST API
    Навыки разработки чат-ботов с учетом сценариев диалогов
    Умение работать с ограничениями по скорости и количеству запросов
    Соблюдение стиля общения бота — дружелюбный и профессиональный
    Критерии приёмки: стабильная работа бота, корректное понимание и ответ на типовые вопросы, соответствие сценарию диалогов, корректный сбор данных

Технологии:
Python
Node.js
GigaChat API
ProxyAPI
Чат-боты
REST API

стоимость: 7000 НКН, дедлайн до 24 июля
